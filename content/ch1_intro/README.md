# Chapter 01: Getting Started with Python

Welcome to Python programming! This chapter covers the fundamentals that will set your foundation.

## What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python emphasizes code readability with its distinctive indentation-based syntax.

## Basic Syntax

Python uses a simple syntax that reads almost like plain English. Lines are terminated by newlines rather than semicolons, and blocks are defined by indentation (typically 4 spaces).

## Comments

Python supports two types of comments:
- **Single-line comments**: Start with `#`
- **Docstrings**: Triple-quoted strings used for documentation
